# VS Code For C# Development

## Need to Add appsettings.json
Structure 

```
{
    "AppSettings" : {
        "Email" : "test.test",
        "Domain" : "@test.com",
        "URL" : "https://test.com/login"
    }
}
```

```
    "Tasks": [
        {
            "Id": 1,
            "Description": "Working on Test1"
        },
        {
            "Id": 2,
            "Description": "Working on Test2"
        }
    ]
}
```
